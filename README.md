# 🎵 Music Catalog — организуй свою музыку

**Music Catalog** — веб-приложение для систематизации музыкальных альбомов и треков с детальной классификацией по тегам: настроение, назначение, национальный колорит, стиль, инструменты и темп.

Сохраняй ссылки на Spotify, добавляй свои теги, ищи треки по любой комбинации фильтров и делись каталогом с друзьями. Без регистрации, без подписок — просто веб-приложение, которое работает в браузере и на GitHub Pages.

## 🎯 Возможности

- 📚 **Каталог альбомов** — структурируй музыку по исполнителям и альбомам
- 🏷️ **Теги** — помечай треки по настроению, назначению, стилю, инструментам, национальному колориту и темпу
- 🔗 **Ссылки на Spotify** — сохраняй и копируй прямые ссылки на треки
- 🎧 **Аудиофайлы** — сохраняй пути к локальным файлам или URL
- 🖼️ **Обложки альбомов** — загружай обложки по ссылке или из файла
- 🔍 **Поиск и фильтрация** — находи треки по любой комбинации тегов
- 📊 **Экспорт** — выгружай каталог в CSV или JSON
- ☁️ **GitHub Pages** — размещай приложение бесплатно в облаке и делись
- 💾 **Автосохранение** — каждое изменение создаёт коммит с описанием

## 🚀 Быстрый старт

**📖 [Полная пошаговая инструкция с коммитами](GITHUB_SETUP.md)**

Кратко:
1. Создайте репо на GitHub → `music-catalog`
2. Загрузьте три файла: `index.html`, `catalog.json`, `README.md`
3. Settings → Pages → Deploy from main branch
4. Ссылка готова: `https://ваш-ник.github.io/music-catalog/`
5. (опционально) Создайте GitHub Token в Settings → Developer settings → Tokens
6. В приложении → ⚙ GitHub → вставьте токен для автосохранения

## 📖 Как пользоваться

### Просмотр
- Левое меню → выбери альбом
- Видишь список треков с тегами
- Клик на трек → раскрывается редактор

### Редактирование трека
1. Нажми на трек в списке
2. Заполни:
   - **Spotify ссылка** → копируй одним кликом
   - **Аудиофайл** → путь или URL
   - **Теги** → выбирай или добавляй свои
3. **Сохранить** → всё сохранится локально и на GitHub

### Фильтр по тегам
1. **🔍 Фильтр по тегам** (верх приложения)
2. Выбирай теги → система сразу показывает совпадения
3. Комбинируй фильтры как хочешь

### Добавление альбома
**+ Альбом** → заполни форму → добавляется в каталог

### Добавление трека
Открой альбом → **+ Добавить трек** → напиши название

## 🏷️ Теги

**Настроение:** спокойное, грустное, радостное, тревожное, медитативное, ностальгическое, тёмное, романтическое, торжественное, загадочное

**Назначение:** фон для работы, концентрация, глубокий фокус, расслабление, сон, медитация, романтика, драйвинг, грусть / отпускание, фоновый

**Нац. колорит:** нейтральный, китайский, аргентинский / танго, балканский, северный / скандинавский, американский / кантри, западный / вестерн, джазовый / американский

**Стиль:** ambient, dark ambient, lofi, classical, folk, electronic, jazz, swing, cinematic, drone, country, tango, indie electronica

**Инструменты:** фортепиано, виолончель, скрипка, гитара, бандонеон, флейта, орган, синтезатор, кларнет, контрабас, струнные, гуцинь

**Темп:** медленный, средний, быстрый

## 📤 Экспорт

- **CSV** → откроется в Excel с всеми тегами
- **JSON** → полная копия для резервной копии или переноса

## 🔐 Безопасность

- **Токен** хранится только в браузере, никуда не отправляется
- **Данные видны** всем, кто знает ссылку, но **редактировать может только вы**
- Для приватного каталога → приватный репо (GitHub Pro)

## 💾 История изменений

Каждое редактирование создаёт автоматический Git коммит:
- `Update track: "название" in "альбом"`
- `Add album: "название" by Исполнитель`
- `Update cover: "альбом"`

История видна в GitHub → Commits
